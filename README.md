# ハクサンリーグ・旧市町村ジムバッジ

白山市の旧8市町村を巡り、NFCタグからジムバッジを集めるデジタルバッジラリーです。

## 🌟 プロジェクト概要

このプロジェクトは、白山市の8つの旧市町村（鶴来、美川、松任、河内、白峰、吉野谷、鳥越、尾口）を巡り、各地でNFCタグ入りピンバッジにタッチすることでジムバッジを獲得できるデジタルバッジラリーです。ポケモンの世界観をモチーフに、各地域の特色を活かしたジムバッジをデザインしました。

## 🎯 機能

- **ジムバッジ獲得機能**: NFCタグからURLアクセスでバッジを自動取得
- **進捗管理**: 取得したバッジの視覚的な表示
- **ジムの物語**: 各ジムの歴史や文化を学べるコンテンツ
- **チャンピオン特典**: 全バッジ取得時の特別コンテンツ
- **ポケモン風マップ**: ドット絵調のゲームライクなマップ表示
- **レスポンシブデザイン**: スマートフォン対応

## 🏗️ 技術仕様

### フロントエンド
- **HTML5**: セマンティックなマークアップ
- **CSS3**: レスポンシブデザイン、アニメーション
- **JavaScript**: スタンプ管理、NFC連携

### ホスティング
- **Netlify**: 静的サイトホスティング
- **GitHub連携**: 自動デプロイ

### データ管理
- **localStorage**: バッジデータの端末内保存

## 📱 対応デバイス

- NFC搭載のスマートフォン（iOS/Android）
- タブレット端末
- デスクトップブラウザ（テスト用）

## 🏆 ハクサンリーグのジムバッジ

1. **鶴来ジム**: クレインバッジ (Crane Badge) - フェアリー/ひこうタイプ
2. **美川ジム**: ラッパバッジ (Trumpet Badge) - みず/はがねタイプ  
3. **松任ジム**: パインバッジ (Pine Badge) - ノーマル/くさタイプ
4. **河内ジム**: ブリッジバッジ (Bridge Badge) - いわ/じめんタイプ
5. **白峰ジム**: ピークバッジ (Peak Badge) - こおりタイプ
6. **吉野谷ジム**: フォレストバッジ (Forest Badge) - くさタイプ
7. **鳥越ジム**: キャッスルバッジ (Castle Badge) - ゴースト/かくとうタイプ
8. **尾口ジム**: フォールバッジ (Fall Badge) - みずタイプ

## 🚀 セットアップ

### 開発環境
```bash
# プロジェクトをクローン
git clone [your-repository-url]

# ローカルサーバーで起動（例：Live Server）
# index.htmlを開く
```

### Netlifyデプロイ
1. GitHubリポジトリを作成
2. Netlifyでサイトを作成
3. GitHubリポジトリと連携
4. 自動デプロイ設定

### NFCタグ設定
各ジムのNFCタグに以下のURLを書き込み：
```
https://your-site.netlify.app/town/{town_name}.html?badge={town_name}
```

例：
- 鶴来: `https://your-site.netlify.app/town/tsurugi.html?badge=tsurugi`
- 美川: `https://your-site.netlify.app/town/mikawa.html?badge=mikawa`
- 尾口: `https://your-site.netlify.app/town/oguchi.html?badge=oguchi`

## 🔧 カスタマイズ

2. `town/`フォルダに新しいHTMLファイルを作成
3. `index.html`にタウンカードを追加

### スタイルの変更
- `styles.css`: メインデザイン
- `town/town-styles.css`: 町ページデザイン

## 🔐 セキュリティ

- スタンプデータは端末のlocalStorageに保存
- 個人情報は取得しません
- 将来的な不正防止対策の拡張性を考慮

## 📊 使用方法

### ユーザー向け

4. 全8市町村でスタンプを集める
5. 特別コンテンツを楽しむ

### 管理者向け

## 🐛 トラブルシューティング

### スタンプが取得されない
- NFCが有効になっているか確認
- URLパラメータが正しいか確認
- ブラウザのJavaScriptが有効か確認

### 表示が崩れる
- ブラウザのキャッシュをクリア
- 最新のブラウザを使用
- JavaScriptエラーをコンソールで確認
