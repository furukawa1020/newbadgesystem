# はくさんバッジクエスト - 白山市NFCスタンプラリー

白山市の旧8市町村を巡り、NFCタグからスタンプを集めるデジタルスタンプラリーです。

## 🌟 プロジェクト概要

このプロジェクトは、白山市の8つの町（鶴来、美川、松任、河内、白峰、吉野谷、鳥越、一里野）を巡り、各町でNFCタグ入りピンバッジにタッチすることでスタンプを獲得できるデジタルスタンプラリーです。

## 🎯 機能

- **スタンプ獲得機能**: NFCタグからURLアクセスでスタンプを自動取得
- **進捗管理**: 取得したスタンプの視覚的な表示
- **町の物語**: 各町の歴史や文化を学べるコンテンツ
- **コンプリート特典**: 全スタンプ取得時の特別コンテンツ
- **レスポンシブデザイン**: スマートフォン対応

## 🏗️ 技術仕様

### フロントエンド
- **HTML5**: セマンティックなマークアップ
- **CSS3**: レスポンシブデザイン、アニメーション
- **JavaScript**: スタンプ管理、NFC連携

### ホスティング
- **Netlify**: 静的サイトホスティング
- **GitHub連携**: 自動デプロイ

### データ管理
- **localStorage**: スタンプデータの端末内保存

## 📱 対応デバイス

- NFC搭載のスマートフォン（iOS/Android）
- タブレット端末
- デスクトップブラウザ（テスト用）

## 🚀 セットアップ

### 開発環境
```bash
# プロジェクトをクローン
git clone [your-repository-url]

# ローカルサーバーで起動（例：Live Server）
# index.htmlを開く
```

### Netlifyデプロイ
1. GitHubリポジトリを作成
2. Netlifyでサイトを作成
3. GitHubリポジトリと連携
4. 自動デプロイ設定

### NFCタグ設定
各町のNFCタグに以下のURLを書き込み：
```
https://your-site.netlify.app/town/{town_name}.html?stamp={town_name}
```

例：
- 鶴来: `https://your-site.netlify.app/town/tsurugi.html?stamp=tsurugi`
- 美川: `https://your-site.netlify.app/town/mikawa.html?stamp=mikawa`

## 🐛 トラブルシューティング

### スタンプが取得されない
- NFCが有効になっているか確認
- URLパラメータが正しいか確認
- ブラウザのJavaScriptが有効か確認

### 表示が崩れる
- ブラウザのキャッシュをクリア
- 最新のブラウザを使用
- JavaScriptエラーをコンソールで確認
