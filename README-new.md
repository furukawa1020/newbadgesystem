# ハクサンリーグ・旧市町村ジムバッジ 🏔️

白山市の8つの旧市町村を巡り、ジムバッジを集める次世代デジタル観光体験アプリです。

## 🎯 観光客爆売れ機能

### 🏅 高度なインセンティブシステム
- **レア度システム**: レジェンダリー（白峰）、レア、アンコモン、コモンの4段階
- **実績システム**: 30種類以上の実績でプレイヤーを動機付け
- **秘密コンテンツ解放**: バッジ獲得で温泉マップ、伝説ストーリー、フォトスポットを段階的に解放
- **限定時間イベント**: 週末ボーナス、夕方特典など期間限定コンテンツ

### 🎮 没入型RPG体験
- **3D地形マップ**: 実際の白山地理に基づく立体的なマップ
- **アイソメトリック風ジム建物**: 各ジムタイプ別の3Dデザイン
- **季節エフェクト**: 冬の雪、秋の落ち葉、天候変化
- **インタラクティブ要素**: 伝説のポケモン、宝箱、野生ポケモンとの遭遇

### 📱 最先端マーケティング機能
- **SNS完全連携**: Twitter、Facebook、Instagram、LINE対応
- **バイラル機能**: QRコード自動生成、グループチャレンジ
- **リピーター特典**: 訪問回数に応じた割引クーポン
- **リアルタイム統計**: 参加者数、完成度の表示
- **口コミシステム**: Google、TripAdvisorレビュー連携

### 🎨 プレミアム世界観
- **ゲーミングフォント**: Orbitron、Zen Kaku Gothic採用
- **ピクセルアート美学**: 本格的なレトロゲーム風デザイン
- **特殊エフェクト**: 花火、画面震え、光る効果、サウンド

### 💰 収益化要素
- **物理バッジ販売**: 500-1000円のNFC付きピンバッジ
- **段階的報酬**: ステッカー→タオル→Tシャツの段階的グッズ展開
- **限定コンテンツ**: プレミアム情報の段階的解放
- **地域連携**: 道の駅での特典、割引システム

## 🚀 使用方法

### 基本操作
1. メインページまたはRPG版ページにアクセス
2. 任意のジムを選択して訪問
3. NFC付きバッジをタップしてバッジ獲得 (`?badge=ジム名`)
4. 進捗確認とシェア機能を活用

### マーケティング活用
1. **SNSシェア**: 自動生成されるシェアボタンでバイラル拡散
2. **グループチャレンジ**: 友達と一緒に競い合い
3. **実績コンプリート**: 全実績獲得で特別な称号
4. **リピーター特典**: 複数回訪問で割引クーポン

### 収益化施策
1. **段階的報酬システム**: 2個→4個→8個で物理グッズ提供
2. **秘密コンテンツ**: バッジ獲得で解放される限定情報
3. **期間限定イベント**: 週末や時間帯別の特別オファー
4. **口コミ促進**: レビュー投稿でさらなる特典
